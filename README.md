# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas 制作的双人坦克对战游戏。

## 🕹️ 游戏预览

打开 `index.html` 即可在浏览器中游玩！

## 🎯 游戏规则

- **双人对战**：蓝方 vs 红方
- **胜负条件**：先获得 **5 分** 的一方获胜
- **得分方式**：击中对方坦克即可得分
- **道具系统**：地图上会随机刷新道具（生命恢复、加速、快速射击）

## ⌨️ 操作说明

| 操作 | 蓝方 P1 | 红方 P2 |
|------|---------|---------|
| 移动 | `W` `A` `S` `D` | `↑` `←` `↓` `→` |
| 射击 | `空格键` | `Enter` |

## 🎨 游戏特色

- 🏗️ **可破坏地形**：砖墙可被子弹摧毁，钢墙不可破坏
- 💥 **粒子特效**：爆炸、射击火花、弹道拖尾
- ❤️ **生命值系统**：每位玩家 100 点生命，每发子弹 25 点伤害
- 🛡️ **复活无敌**：阵亡后 2 秒复活，复活后有短暂无敌时间
- 🎁 **道具系统**：
  - `+` 生命恢复（绿色）
  - `>` 加速移动（蓝色）
  - `R` 快速射击（橙色）
- 🎨 **精致画面**：渐变配色、光影效果、履带纹理

## 🚀 快速开始

```bash
# 直接用浏览器打开
open index.html

# 或使用任意 HTTP 服务器
python -m http.server 8000
# 访问 http://localhost:8000
```

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- 无需任何依赖或构建工具
